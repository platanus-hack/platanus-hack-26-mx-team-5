## Overview
This page outlines the supported versions of the Meta AI app and glasses firmware for each release of the Meta Wearables Device Access Toolkit.

## Use this page with AI

Copy this prompt into your AI coding tool to check version compatibility before changing code:

```text
Use https://wearables.developer.meta.com/docs/develop/dat/version-dependencies/, then use the Wearables MCP endpoint https://mcp.developer.meta.com/wearables to call search_dat_docs for current DAT version dependency guidance. Inspect my app SDK version, target devices, Meta AI app version, and glasses firmware assumptions first. Tell me whether the environment is compatible, what must be upgraded, and which local or device checks should pass before I debug app code.
```
